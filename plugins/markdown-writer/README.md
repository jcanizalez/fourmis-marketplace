```bash
claude plugin add --from https://github.com/jcanizalez/fourmis-marketplace/plugins/markdown-writer
```

## Overview

Professional writing toolkit — blog posts, technical docs, SEO content, readability analysis, content outlining, and systematic editing. 6 skills, 3 commands, 1 editor agent. No dependencies.

## Skills

| Skill | Activates when... |
|-------|-------------------|
| `blog-writing` | Write compelling blog posts with proven structure — hook openings |
| `content-outlining` | Create structured content outlines for articles |
| `editing-checklist` | Systematic editing and proofreading checklist for polishing written content. Use... |
| `readability-optimization` | Analyze and improve text readability using Flesch-Kincaid scores |
| `seo-writing` | Optimize written content for search engines — keyword placement |
| `technical-writing` | Write clear technical documentation |

## Commands

| Command | Description |
|---------|-------------|
| `/draft` | Generate a structured first draft from a topic — choose format (blog, tutorial, docs) and get a complete draft with proper structure |
| `/outline` | Create a structured content outline for any format — blog posts, tutorials, documentation, comparisons, or listicles |
| `/readability` | Analyze text readability — Flesch score, sentence length, passive voice, word complexity, and actionable improvement suggestions |

## Agents

### editor
Use this agent to review and improve written content — performs structural, clarity, and polish editing passes with specific, actionable suggestions.
