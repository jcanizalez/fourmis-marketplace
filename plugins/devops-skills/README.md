| Command | Description |
|---------|-------------|
| `/deploy-check` | Run a pre-deployment safety checklist — verify code, infrastructure, rollback plan, and timing |

| `/infra-review` | Review infrastructure configuration — Dockerfiles, K8s manifests, CI/CD pipelines, and security posture |

## Agents

### incident-responder
Use this agent for autonomous incident investigation — analyzes logs, checks recent changes, identifies root causes, and suggests mitigations for production issues.

### infra-reviewer
Use this agent for comprehensive infrastructure configuration audits — reviews Dockerfiles, K8s manifests, CI/CD workflows, and security posture against best practices.
